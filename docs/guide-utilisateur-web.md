# Guide utilisateur web

## 1. Présentation

L'application web permet de consulter les événements, créer un compte, se connecter, s'inscrire à un événement et accéder à des espaces spécifiques selon le rôle de l'utilisateur.

## 2. Accès à l'application

1. Ouvrir l'application web dans un navigateur.
2. Depuis la page d'accueil, consulter la liste des événements disponibles.
3. Utiliser les boutons de navigation pour se connecter ou créer un compte.

## 3. Fonctions principales

### Consulter les événements

- La page d'accueil affiche la liste des événements.
- Chaque carte permet d'accéder au détail d'un événement.

### Créer un compte

1. Ouvrir la page d'inscription.
2. Saisir le nom, l'email et le mot de passe.
3. Valider le formulaire.

### Se connecter

1. Ouvrir la page de connexion.
2. Saisir l'email et le mot de passe.
3. Valider pour accéder à l'espace personnel.

### Consulter le détail d'un événement

- Le détail présente le titre, la description, la date, le lieu, le prix et les informations utiles.
- Depuis cette page, l'utilisateur peut s'inscrire selon ses droits et l'état de l'événement.

### Gérer son espace utilisateur

- Un utilisateur connecté peut accéder à son tableau de bord.
- Il peut consulter ses informations et ses inscriptions selon les fonctionnalités disponibles.

### Espace organisateur

- Un organisateur peut créer un événement.
- Il peut modifier ou supprimer les événements qu'il a créés.

### Espace administrateur

- Un administrateur peut gérer les utilisateurs.
- Il peut consulter les événements et effectuer des actions de gestion avancées.

## 4. Déconnexion

- La déconnexion met fin à la session et redirige l'utilisateur vers une zone publique de l'application.

## 5. Prérequis d'utilisation

- Disposer d'une connexion internet ou d'un accès réseau au serveur.
- Utiliser un navigateur compatible comme Google Chrome ou Mozilla Firefox.
