# Guide utilisateur mobile

## 1. Présentation

L'application mobile permet de consulter les événements, se connecter, s'inscrire à un événement et gérer certaines actions utilisateur depuis un téléphone.

## 2. Accès à l'application

1. Lancer l'application mobile depuis Expo Go ou depuis la version installée.
2. Attendre le chargement initial.
3. Accéder à l'écran de connexion ou d'inscription si aucune session n'est active.

## 3. Fonctions principales

### Se connecter

1. Saisir l'email et le mot de passe.
2. Valider le formulaire.
3. L'application ouvre ensuite l'écran principal.

### Créer un compte

1. Ouvrir l'écran d'inscription.
2. Saisir les informations demandées.
3. Valider le formulaire pour créer le compte.

### Consulter les événements

- L'écran principal affiche la liste des événements.
- Chaque carte donne accès au détail d'un événement.

### Voir le détail d'un événement

- L'utilisateur peut consulter la date, le lieu, le prix, la description et les éventuelles photos.

### S'inscrire à un événement

- Depuis la fiche d'un événement, l'utilisateur peut confirmer son inscription.
- Si l'utilisateur est déjà inscrit, il peut aussi se désinscrire.

### Créer ou modifier un événement

- Un utilisateur autorisé peut accéder au formulaire de création d'événement.
- Il peut ajouter ou modifier les informations de l'événement.

### Gérer son compte

- L'application permet d'afficher les informations liées au compte et les inscriptions en cours.
- L'utilisateur peut se déconnecter à tout moment.

### Fonctions administrateur

- Si l'utilisateur possède un rôle administrateur, un espace supplémentaire permet d'accéder à des informations de gestion.

## 4. Prérequis d'utilisation

- Disposer d'un smartphone Android ou iOS.
- Être sur le même réseau que le serveur en environnement local.
- Disposer d'une connexion réseau valide.
