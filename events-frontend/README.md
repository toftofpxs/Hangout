- Identifiant : [à préciser si besoin]

## Développement local

1. npm install
2. npm run dev

Configurer `VITE_API_BASE_URL` dans `.env` si nécessaire (exemple : http://localhost:3000/api).

## Remarques

En cas de problème d'accès, veuillez vérifier la connexion internet ou contacter le développeur.
